# PokePatch

A barebones [Next.js](https://nextjs.org/) + React + Tailwind CSS site, deployable to GitHub Pages.

## Quick Start

```bash
cd pokepatch-website
npm install
npm run dev
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

## Tech Stack

- React
- Next.js (static export)
- Tailwind CSS
- gh-pages
